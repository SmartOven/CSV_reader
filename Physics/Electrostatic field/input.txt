Документация к работе:
После каждой переменной (например, "collision_type=" должен стоять хотя бы один пробел, количество не важно, главное - не меньше одного)
Чтобы изменить стартовые параметры объектов, нужно изменять:
	1) Массу:
		"small_cube_mass=" для изменения массы маленького кубика,
		"big_cube_mass=" для изменения массы большого кубика
	2) Скорость:
		"small_cube_start_velocity=" для изменения начальной скорости маленького кубика,
		"big_cube_start_velocity=" для изменения начальной скорости большого кубика
	3) Тип соударения кубиков:
		"collision_type=" для изменения типа соударения (0 - абсолютно упругое, 1 - абсолютно неупругое)
Можно также изменить (не влияет на соударения кубиков):
	1) Начальную координату кубика:
		"small_cube_start_position=" для изменения начальной координаты маленького кубика,
		"big_cube_start_position=" для изменения начальной координаты большого кубика
	2) Размеры мира:
		"world_size_x=" для изменения ширины "мира" (левая стена имеет координату 0, правая - координату world_size_x)
	3) Размеры окна программы:
		"window_size_x=" для изменения ширины окна программы
		"window_size_y=" для изменения высоты окна программы
